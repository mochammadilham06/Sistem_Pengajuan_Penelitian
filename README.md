### How to Run?

1. Clone this Project
2. On your command, type cd Sistem_Pengajuan_Penelitian
3. type `code .`
4. On your Code editor, open terminal and then type `npm install`
5. Run the project by typing `php artisan serve`
6. Happy Testing
